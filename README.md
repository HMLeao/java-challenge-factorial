# Desafio

## Cálculo de fatorial

Dado o método findFactorial(int), que recebe um dado numérico inteiro como parâmetro,
retorne seu fatorial. Por exemplo, se a entrada for 5, o método deve retornar
(5*4*3*2*1) = 120. Obs.: Lembre-se que o 0! = 1

Exemplo 1:
- entrada: 4
- saída: 24

Exemplo 2:
- entrada: 8
- saída: 40320
